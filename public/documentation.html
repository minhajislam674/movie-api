myFlix Movie API Documentation

Brief Info

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoint description Method type Endpoint URL Request body data format Response body data format
Return a list of ALL movies to the user GET /movies - A JSON object holding data about all the movies
Return a single movie by title to the user GET /movies/:title - A JSON object holding data about a single movie containing title,description, genre, director
Return data about movie genre GET /movies/genre/:genreName - A JSON object containing genre name and description
Return data about a director by name GET /movies/director/:directorName - A JSON object containing name, bio and birth year of the director
Add new user/Allow new users to register POST /users A JSON object holding data about the user to add A JSON object holding data about the added user including an unique id
Allow users to update their user info (username) PUT /users/:id A JSON object containing the updated name A JSON object with the new info, others remain unchanged
Allow users to add a movie to their list of favorites POST /users/:id/:movieTitle - A text message indiciating favorite movie was succesfully added/td>
Allow users to delete a movie from their list of favorites DELETE /users/:id/:movieTitle - A text message indiciating favorite movie was succesfully removed/td>
Allow existing users to deregister DELETE /users/:id - A text message indiciating user info was removed/td>
