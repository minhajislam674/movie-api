myFlix Movie API Documentation

Brief Info

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Key Features

  1. Return a list of ALL movies to the user
  2. Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  3. Return data about a genre (description) by name/title (e.g., “Thriller”)
  4. Return data about a director (bio, birth year, death year) by name
  5. Allow new users to register
  6. Allow users to update their user info (username, password, email, date of birth)
  7. Allow users to add a movie to their list of favorites
  8. Allow users to remove a movie from their list of favorites
  9. Allow existing users to deregister
Business Logic HTTP Method type Endpoint URL Request body data format Response body data format
Return a list of ALL movies to the user GET /movies - A JSON object holding data about all the movies
Return a single movie by title to the user GET /movies/:Title - A JSON object holding data about a single movie containing title,description, genre, director
Return data about movie genre GET /movies/genres/:Name - A JSON object containing genre name and description
Return data about a director by name GET /movies/directors/:Name - A JSON object containing name, bio and birth year of the director
Add new user/Allow new users to register POST /users A JSON object holding data about the user to add A JSON object holding data about the added user including an unique id
Allow users to update their user info (username) PUT /users/:Username A JSON object containing the updated name A JSON object with the new info, others remain unchanged
Allow users to add a movie to their list of favorites POST /users/:Username/movies/:MovieID - A JSON object with the new user info which also includes "FavoriteMovies": ["635932bafb472953a30232d1"]
Allow users to delete a movie from their list of favorites DELETE /users/:Username/movies/:MovieID - A JSON object with the new user info. After deletion, it will look like this. "FavoriteMovies": [""]
Allow existing users to deregister DELETE /users/:Username - A text message indiciating user info was removed
